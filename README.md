<h2>📌 Project Overview</h2>

<p>
This project presents an <b>IPL Data Analysis Dashboard</b> created using 
<b>Power BI</b> and <b>Excel</b> to explore team and player performance. 
The dashboard provides insights into match results, top players, and overall season trends 
through interactive visualizations.
</p>

<hr>

<h2>📊 Key Insights</h2>

<ul>
<li>Team win analysis across multiple IPL seasons</li>
<li>Top run scorers and leading wicket takers</li>
<li>Match result trends and performance comparisons</li>
<li>Player impact analysis</li>
<li>Interactive filters for better data exploration</li>
</ul>
